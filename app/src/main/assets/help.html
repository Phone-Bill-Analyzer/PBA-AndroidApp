Getting Started

Thank you for downloading this app.

Now, you have the option to really understand your phone bill and analyze the usage also. Just upload your mobile phone bill and start your analysis.

Upload Mobile Bill:

On the main screen, select option: New Bill and upload new bill.
Currently we are supporting only Airtel Postpaid bill, Vodafone Postpaid bill and Reliance Post Paid bills in PDF formats. If your pdf file is password protected, then please enter the password also.
Note :
  • If you are unable to browse files, please install a good file browser app. We recommend using ES file explorer app.
  • In the file browser screen, activate 'Display advanced devices' option to see all files and folders

Analysis

After you have successfully uploaded your bill, you can see your bill details on the main screen. Select your bill to start analyzing. As of now, you can see the following analysis:
  • Itemised bill - Summary
  • Itemised bill - Details
  • Top 5 callers by Name
  • Call costs by Contact groups
  • Contacts without names
  • Contacts not belonging to Group
In future, we are planning to show more analytics.

Advanced Analysis

In the free version of the app, you can use features of Basic Analysis only. The below features are available only for premium users.
Please upgrade to premium user to use advanced analysis.
  • Compare your monthly bill usage
  • Download your itemized bill as CSV file for detailed analysis

Contact Groups

You can create contact groups for e.g. Family, Friends, Co-Workers etc.
Important Points to note:
  • You can assign a contact to multiple groups.
  • You can update contact names, groups and group members after uploading the bill also. Hence, the bill analysis will continue to show old names and old groups.
  • If you have modified your contacts and groups, then please reload the contact details from main screen.

Download as CSV File

Premium users can download their itemized statements as CSV files for analysis on their computers.
The file will be downloaded in following location: Android/Data

Limitations

  • As of now, we support only Airtel, Vodafone, Reliance & Tata Docomo Post Paid bills. We will support more in future
